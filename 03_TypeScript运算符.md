# 第三章 `TypeScript`运算符

## 一、运算符的概述

运算符也称为操作符，用来实现赋值（=）、算数运算、比较等功能的符号

## 二、算数运算符

### 1、概述

算数运算符包含：加（+）、减（-）、乘（*）、除（/）

用来实现两个数之间的计算

## 2、加法的其他作用

- +号，不仅可以使用加法计算，也可以进行字符串的拼接
- 加号两边只要又一遍是字符串，就进行字符串拼接

### 3、其他运算符

除了加号外，其他算术运算符只能跟数字类型一起使用

其他方式：把字符串前边加一个+。2 - +'1'  // 1

## 三、赋值运算符

赋值运算符：将等号右边的值赋值给它左边的变量，比如：等号（=）。

- 顺序：先计算等号右边的代码，再将计算结果赋值给等号左边的变量
- 加等于+=、减等于-+、乘等于*=、除等于/=。

## 四、自增自减运算符

自增（++）运算符是+=1的简化形式；自减（--）运算符是 -=1 的简化形式

age++

- 作用：自增运算符是让变量的值加1，自减运算符是让变量的值减1，自己能实现加1和减1

## 五、比较运算符

用于比较两个数据的值，并返回其比较结果，结果为布尔类型

比较运算符包含6个：

- 大于 >
- 大于等于 >=
- 小于 <
- 小于等于 <=
- 等于 ===
- 不等于 !==

## 六、逻辑运算符

通常用于布尔类型的计算，并且结果也是布尔类型

- 与（逻辑与） &&，同时为true，结果才为true，否则为false
- 或（逻辑或）||，只要有一边为true，结果就为true，只有都为false，才为false
- 非（取反 / 逻辑非）！，true变false，false变true
